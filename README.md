# Amamiya-SD

后备库房——储存不是很重要的资源等  
教程——教程部的活动记录，部分文字教程或者小技巧也可在此刊载  
服务器——服务器历史整合包发布和服务器历史版本记录  
档案馆——存放小组和服务器的历史  
永恒乐园本地化——汉化记录和部分汉化文件存档和发布的地方  
视频部——关于B站公众号的事情
